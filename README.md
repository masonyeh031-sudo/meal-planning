# Meal Planning

每日飲食份數與營養素計算網站原型。

## 功能

- 輸入身高、體重、年齡、性別、活動量、飲食目標
- 自動估算每日建議熱量與 BMI
- 自動產生六大類飲食份數
- 即時計算 CHO / PRO / FAT 與熱量
- 可手動微調份數
- 匯出 PDF、JPG、Google 試算表 CSV、Excel

## 本機執行

```bash
npm install
npm run dev
```

本機預設網址：

```bash
http://localhost:3000
```

也保留：

```bash
http://localhost:3000/nutrition-calculator
```
